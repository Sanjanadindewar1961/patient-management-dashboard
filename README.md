# 🏥 Patient Management Dashboard

## 📌 Overview

This is a Patient Management Dashboard built using React and TypeScript. The application allows users to manage patient records efficiently with features like adding, viewing, and searching patients.

---

## 🚀 Features

* Add new patients with validation
* View all patients in a structured table
* Search patients by name or diagnosis
* Display total patient count
* Performance optimization using useMemo

---

## 🛠️ Tech Stack

* React (Functional Components & Hooks)
* TypeScript
* Vite

---

## 🧠 Concepts Used

* CRUD Operations (Create, Read)
* Modular Architecture (components, hooks, types)
* State Management using custom hooks
* Performance Optimization using useMemo
